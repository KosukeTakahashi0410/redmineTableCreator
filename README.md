### 概要説明
Redmineで表を作るときに面倒だったので作成  
ヘッダー・表要素の必要な数を記載して作成ボタンを押下するとクリップボードにコピーされる  
また、テンプレートボタンを押下すれば表のテンプレートが取得できる（現在はデバイスとテストケースのみ）  
現状Github経由でclone、開発者として使うしかないので、テンプレート箇所を適当に変える（もしくはinputを増やす）などすれば、任意の値をコピーできます
